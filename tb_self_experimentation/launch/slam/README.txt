1- Configure the turtlebot ip and the host ip in the .bashrc files
2- export the map or set it in the .bashrc file
3- Launch the slam_tb using ssh in the turtlebot computer
4- Launch the slam_basestation in the basestation computer
5- Use RVIZ to estimate initial position and to give navigation points
